# MiniStore

MiniStore is a simple frontend e-commerce project built using **HTML, CSS, and JavaScript**.

## Features
- Fetches products from DummyJSON API  
- Responsive navbar with search  
- Pagination (Prev / Next)  
- Auto-load products on page load  
- Clean product card layout  

## Tech Used
- HTML  
- CSS  
- JavaScript  
- DummyJSON API
